# 队列 queue
**先进先出** FIFO  
只允许在表的一端进行插入，而在另一端删除元素。  
允许插入的一端称为队尾（rear），允许删除的一端则称为队头（front）  
*保存的是队头元素的下标值，队尾元素下一个位置的下标值*
### 队列的顺序结构-初始化
~~~
#define MAXSIZE 100
typedef int ElemType;
typedef struct{
  ElemType data[MAXSIZE];
  int front;
  int rear;
}Queue;
void initQueue(Queue *Q){
  Q->front = 0;
  Q->rear = 0;
}
~~~
