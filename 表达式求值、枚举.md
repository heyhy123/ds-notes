# 枚举
将变量的值一一列举出来，变量的值只限于列举出来的值的范围内  
在枚举中列出的每一个值，称为枚举元素  
每一个枚举元素由系统定义了一个用序号表示的数值，从0开始，分别为0,1,2...  
语法:
~~~
enum 枚举名
{
   枚举元素......
}
~~~
    #include<stdio.h>
    enum weekday{
        mon,tue,wed,thu,fri,sat,sun
    };
    int main(){
        enum weekday a;
        a=mon;
        enum weekday b;
        b=tue;
        printf("%d\n",a);
        printf("%d\n",b);
        return 0;
    }  //输出0,1
    //若想从1开始计数，枚举定义时mon=1
在上边代码中，每次定义weekday类型数据时，都需要加enum，有点麻烦，可以使用typedef 
    
    tepedef enum{
        mon,tue,wed,thu,fri,sat,sun
    }weekday;
