## 顺序查找
应用范围：  
1.顺序表或线性链表表示的静态查找表  
2.表内元素之间无序  

***代码如下***   
*1.search函数*

    #include<stdio.h>

    int search(int* data,int len,int value){
        for(int i = 0;i < len;i++){
            if(data[i] == value){
                return i;
            }
        }
        return -1;
    }

*2.main函数*

    int main(int argc,char const *argv[]){
        int arr[] = {15,54,76,6,9};
        int len = sizeof(arr) / sizeof(arr[0]);
        int pos = search(arr,len,54);
        printf("%d\n",pos);
        return 0;
    }

**改进：把待查关键字key存入表头（“哨兵”），从后向前逐个比较可免去查找过程中每一步都要检测是否查找完毕，加快速度。**  
***代码如下***  

    //在顺序表L中查找值为e的数据元素
    int search(SSTable ST,KeyType key){
        ST.R[0].key = key;                              //key存入顺序表表头
        for(i = ST.length;ST.R[i].key != key;--i);      //不用检测是否查找完毕，只用比较是否匹配
        return i;                                       //若成功返回其位置信息，否则返回0
    }

## 折半查找（二分查找）
**前提：查找表必须是有序的**  
### 非递归算法
<img width="284" height="191" alt="image" src="https://github.com/user-attachments/assets/296aff17-005e-49fe-ae93-faf1ccbd131f" />  

***代码如下***  
*1.binary_search函数*

    #include<stdio.h>

    int binary_search(int* data,int len,int value){
        int low = 0;
        int high = len-1;
        int mid;
        while(low <= high){
            mid = (low + high) / 2;
            if(value > data[mid]){
                low = mid + 1;
            }
            else if(value < data[mid]){
                high = mid - 1;
            }
            else{
                return mid;
            }
        }
        return 0;
    }
